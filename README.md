## Usage

1. Open your Jetbrains IDE
2. Open `Plugins` > `Marketplace` > `Search for "Roxo Theme"` and install the plugin.
3. Open `Settings/Preferences` > `Appearance & Behavior` > `Appearance` > `Theme` > `Roxo` to apply Theme.
